Base

A comprehensive front-end framework for designing and developing web applications

Goals

Unify design and development

Base offers a shared resource for designers, front-end engineers and back-end engineers to align on proper styling, markup, and interactivity. Since Base exists outside of our product development stack, designers and developers can tweak, adjust, and perfect the UI of our product without increasing the volatility of our code base.

Reduce design and development time

Rather than creating new features from a blank canvas, both designers and developers can use existing components, patterns and rules from this centralized design system. Not only will this reduce the amount of time required to create new features, it will also reduce QA testing as we build with tried-and-true code.
