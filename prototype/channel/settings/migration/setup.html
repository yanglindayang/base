Users File Format

Column Heading Values Description
EXTERNAL_ID 255 max character limit Required The primary key of the user in the existing system.
EMAIL_ADDRESS 255 max character limit Required The email address of the user.
FIRST_NAME 255 max character limit Optional The first name of the user.
LAST_NAME 255 max character limit Optional The last name of the user.
ACTIVATED TRUE, FALSE Optional A boolean value which determines if the user is activated on AppDirect. Set to TRUE if appdirect will not be the identity provider. Set to FALSE if appdirect will be the identity provider.
CURRENCY Required The currency of the user.

Companies File Format

Column Heading Values Description
EXTERNAL_ID 255 max character limit Required The primary key of the company in the existing system.
NAME 255 max character limit Optional The name of the company.
STREET1 255 max character limit Optional The first line of the street address of the company.
STREET2 255 max character limit Optional The second line of the street address of the company.
CITY 255 max character limit Optional The city of the company address.
STATE 255 max character limit Optional The state of the company address.
ZIP 255 max character limit Optional The zip of the company address.
COUNTRY 255 max character limit Optional The country of the company address.
PHONE Optional The phone number of the company.
BILLING_PHONE_NUMBER 255 max character limit Optional The billing phone number of the company.
BILLING_STREET1 255 max character limit Optional The first line of the billing street address of the company.
BILLING_STREET2 255 max character limit Optional The second line of the billing street address of the company.
BILLING_CITY 255 max character limit Optional The city of the company billing address.
BILLING_STATE 255 max character limit Optional The state of the company billing address.
BILLING_ZIP 255 max character limit Optional The zip of the company billing address.
BILLING_COUNTRY 255 max character limit Optional The country of the company billing address.
CREATED_BY_EXTERNAL_ID 255 max character limit Optional The external identifier of the user who created the company. Typically the sys admin of the company.
CHANNEL_ACCESS TRUE, FALSE Optional
RSSR_GRANTING TRUE, FALSE Optional Allows the company to see the channel tab. This is required for companies that contain ssr/rssr memberships.
BILLING_HANDLE 255 max character limit Optional
BILLING_HANDLE_TYPE 255 max character limit Optional

Memberships File Format

Column Heading Values Description
USER_EXTERNAL_ID String. 255 max character limit. Required The unique id of the user in the preexisting system. This id is the same as the EXTERNAL_ID from the user migration.
COMPANY_EXTERNAL_ID String. 255 max character limit. Required The unique id of the company in the preexisting system. This id is the same as the EXTERNAL_ID from the company migration.
BILLING_ADMIN Boolean. TRUE, FALSE Optional Determines if the user is a billing admin. If set to TRUE the user will be a billing admin. If set to FALSE the user will not be a billing admin.
SYS_ADMIN Boolean. TRUE, FALSE Optional Determines if the user is a system administrator. If set to TRUE the user will be a system administrator. If set to FALSE the user will not be a system administrator.
SALES_SUPPORT Boolean. TRUE, FALSE Optional TRUE if user is sales support role. FALSE if user is not sales support role.
RESTRICTED_SALES_SUPPORT Boolean. TRUE, FALSE Optional Determines if the user is a restricted sales support. If set to TRUE the user will be a restricted sales support. If set to FALSE the user will not be a restricted sales support.
BILLING_HANDLE Optional The 3rd party billing handle of the user. Only required if current billing provider will be used on AppDirect.
BILLING_HANDLE_TYPE Optional The type of billing handle associated with a user.
MOSI_USER_NAME String. 64 max character limit. Optional The MOSI username of the user who belongs to the company.
BILLING_PHONE_NUMBER Optional
BILLING_STREET1 Optional
BILLING_STREET2 Optional
BILLING_CITY Optional
BILLING_STATE Optional
BILLING_ZIP Optional
BILLING_COUNTRY Optional

Resellers File Format

Column Heading Values Description
RSSR_EXTERNAL_ID Required The external identifier of the rssr. This id is defined as EXTERNAL_ID in the Users tab.
COMPANY_EXTERNAL_ID Required The external identifier of the company. This id is defined as EXTERNAL_ID in the Companies tab.

Company Entitlements File Format

Column Heading Values Description
EXTERNAL_ID String. 255 max character limit. Required The unique id of the purchase from the preexisting system.
EDITION_PRICING_UUID String. 36 max character limit. Required The code of the edition purchased. This edition code will map back to a code from the preexisting system.
COMPANY_EXTERNAL_ID String. 255 max character limit. Required The unique id of the company from the preexisting system. This is the same id defined as EXTERNAL_ID in the company migration.
PURCHASED_BY_EXTERNAL_ID String. 255 max character limit. Required The unique id of the user from the preexisting system. This is the same id defined as EXTERNAL_ID in the user migration.
VENDOR_ENTITLEMENT_ID String. 255 max character limit. Optional The unique id of the purchase defined by the app vendor.
PRICING_UNIT String. [USER | GIGABYTE | HOUR | NOT_APPLICABLE] Required The unit description of the edition.
UNITS Number. 20 digits max. Max of 10 digits before the decimal, and max of 10 digits after the decimal. Optional The number of pricing units in the order.
DISCOUNT_CODE String. 255 max character limit. Optional The discount code.
ADDON_PRICING_UUID String. 36 max character limit. Optional The code (uuid) of the addon that was purchased. This addon code will map to the code from the original system.
START_DATE Format: YYYY-MM-DDThh:mm:ssTZD Optional The timestamp the purchase was made. ie. 1997-07-16T19:20:30+01:00
AUTO_RENEW Boolean. TRUE, FALSE Optional Is the contract auto renew?

User Entitlements File Format

Column Heading Values Description
COMPANY_ENTITLEMENT_EXTERNAL_ID String. 255 max character limit. Required The unique id of the company entitlement. This is defined as EXTERNAL_ID for active orders.
USER_EXTERNAL_ID String. 255 max character limit. Required The unique id of the user from the preexisting system. This is the same id defined as EXTERNAL_ID in the user migration
VENDOR_ENTITLEMENT_ID Optional
STATUS Optional

MOSI Subscriptions File Format

Column Heading Values Description
SUBSCRIPTION_ID String. 36 max character limit. Required MOSI Subscription ID.
COMPANY_ENTITLEMENT_EXTERNAL_ID Required Unique ID associated with the company entitlement. This is the EXTERNAL_IDENTIFIER defined in active order.
MOSI_CUSTOMER_ID Required The MOSI customer ID associated with the subscription.
OFFER_ID String. 36 max character limit. Required
BILLING_ID String. 36 max character limit. Required
SUBSCRIPTION_STATE String. SUBSCRIPTION_STATUS_PROVISION_PENDING | SUBSCRIPTION_STATUS_ORDER_PENDING | SUBSCRIPTION_STATUS_ACTIVE | SUBSCRIPTION_STATUS_DISABLED | SUBSCRIPTION_STATUS_DEPROVISIONED | SUBSCRIPTION_STATUS_INGRACEPERIOD Optional State of the subscription.
QUANTITY Integer. Required Number of seats purchased.
START_EFFECTIVE_DATE Required
START_EFFECTIVE_DATE Required
END_EFFECTIVE_DATE Required
AUTO_RENEW Boolean. TRUE, FALSE Optional Is auto renew set for subscription? Default is TRUE.
PARENT_SUBSCRIPTION_ID Optional The parent subscription ID. Add-on ish. This product needs another product to be owner first.
CONVERTED_SUBSCRIPTION_ID Optional The converted subscription ID.,Free to paid. The ID of the free ID.
SOURCE_SUBSCRIPTION_ID Optional The source subscription ID. Migrated from app A to app B. MOSI transition.

MOSI Domains File Format

Column Heading Values Description
CUSTOM_DOMAIN String. 255 max character limit. Required Custom domain name for the account.
MOSI_CUSTOMER_ID String. 36 max character limit. Required Unique ID to identity the customer with Microsoft.
DOMAIN_STATUS String. CREATED | PENDING_VERIFICATION | VERIFIED | FEDERATED Required Status of the custom domain.
PRIMARY Boolean. TRUE, FALSE. Optional Boolean determining if this domain is the primary domain. Default value is false.
