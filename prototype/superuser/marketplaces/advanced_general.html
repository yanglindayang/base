Features

Marketplace will be accessible only through API's. End-users will not be able to see the marketplace UI, including the checkout and account sections. Only channel admins and developers will be able to log in.
If enabled, users will be redirected to /listing when they visit /home.
Currently, only AppDirect has security seals available. Security seals require custom html per marketplace.

User Accounts

Non-authenticated users will be automatically redirected to login.
After signing up, users will see a success message informing them that their request was sent. They will receive the account activation email when a channel admin approves their account request.
If enabled, allows Marketplace Support to impersonate user without user permission.
If enabled, users will have the option of logging in via selected providers instead of a marketplace account.
