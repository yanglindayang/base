The optional discount logo appears when a customer browses all available discounts on /deals.

If no application is selected, the discount will be applied to all applications in your marketplace.

If no edition is selected, the discount will be applied to all editions in the application.

If no payment plan is selected, the discount will be applied to all payment plans in the edition.

Customers see the description in their order and invoice details.

Choose a discount restriction if you want to block users from redeeming this discount more than once. When a discount is restricted per company, once any user within that company redeems the discount, the discount is no longer valid for any other users in that company.

The maximum number of times the discount can be redeemed. If the discount is automatically applied, it will no longer appear in the marketplace once the max redemptions has been reached.

The end date of when the discount is valid. If an automatically applied discount, the customer will see this in the discount details.

The number of billing cycles the discount is applied to if the discounted subscription is recurring. If left blank, the discount will be applied to all billing cycles.
